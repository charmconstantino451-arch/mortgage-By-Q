# Modern & 3D Design Inspiration: Mortgage & Fintech Niche

This report covers the most visually forward-looking websites in the mortgage and fintech-adjacent space — brands that go beyond the generic blue/white bank aesthetic and push modern design principles.

---

## Summary: Design Tiers in the Mortgage Space

| Tier | Description | Examples |
| :--- | :--- | :--- |
| **Tier 1 — 3D/Interactive Pioneers** | Use actual WebGL, Spline 3D, or scroll-driven animation to create immersive experiences | Habito, Movement Mortgage, cutting-edge design agency concepts |
| **Tier 2 — Premium Modern Lifestyle** | Bold palettes, warm imagery, glassmorphism, strong visual storytelling | Tomo Mortgage, Own Up, Mojo Mortgages |
| **Tier 3 — Clean Fintech / Dark Mode** | Minimal, high-contrast, dark backgrounds, gradient accents, premium SaaS feel | Figure.com, Upstart, Credible |

## 1. Figure.com — "America's #1 Non-Bank HELOC Lender"
**URL**: [figure.com](https://www.figure.com/)  
**Design Tier**: Tier 3 — Clean Fintech / Dark Mode

### Design Breakdown
*   **Color Palette**:
    *   **Backgrounds**: `#F2F2F2` (light mode) / `#1A1B22` (dark mode — deep space charcoal)
    *   **Accent/Action**: Warm gradient image backgrounds (`gradient-1.jpg`)
    *   **Text**: Neutral charcoal `#1A1B22`
*   **Typography**: `Instrument Sans` (modern geometric sans-serif) for body + `Bagel Fat One` (expressive display font) for feature headings
*   **Key Features**:
    *   **Dark Mode native**: The page meta includes dark/light color scheme switching, giving it a premium, tech-forward feel by default
    *   **Astro Framework**: Built with high-performance Astro.js + React islands for fast, SEO-optimized rendering
    *   **Gradient product mockups**: Hero section uses color-washed gradient JPEGs as the background behind a product phone screenshot
*   **Vibe**: Deep tech, clinical precision, and premium. Feels like a blockchain/fintech startup that also does mortgage products. Lean and minimal, but commanding.
*   **Primary CTA**: "Find my HELOC rate" (persistent in header)

### What Makes It Stand Out
Figure is one of the few mortgage-adjacent brands that has actually **adopted a dual dark/light aesthetic**. It uses the language of premium SaaS (deep dark neutrals, high-contrast type, data-driven illustrations) to make financial products feel technologically sophisticated.

### Takeaway for Mortgage by Q
- Adopt `Instrument Sans` or a similar modern geometric typeface for body text.
- Consider a gradient-washed hero section to create visual depth without needing actual 3D.
- Offer a light/dark mode toggle as a premium UI signal.

## 2. Own Up (ownup.com) — "Bringing Clarity to an Outdated Mortgage Industry"
**URL**: [ownup.com](https://www.ownup.com/)  
**Design Tier**: Tier 2 — Premium Modern Lifestyle

### Design Breakdown
*   **Color Palette** (from CSS source):
    *   `aloe-100`: `#3E6D6A` (deep sage/teal — primary buttons, sliders)
    *   `atlantic-100`: `#053C53` (deep ocean navy — headers)
    *   `mango-100`: `#FFE485` (warm yellow — vibrant accent)
    *   `mint-100`: `#B9DDCC` (soft green — backgrounds)
    *   `moss-100`: `#EDF5F1` (off-white green — section fills)
    *   `fog-20`: `#F8F9F9` (lightest neutral)
    *   `charcoal-100`: `#1D1D1F` (apple-style near-black)
*   **Typography**: `Aktiv Grotesk` (body) + `P22 Mackinac` (display serif for warmth) — a premium font pairing
*   **Key Design Technique**: Gradient from `mint-80` to `atlantic-100` used on hero sections, creating a sweeping ocean-to-deep-forest effect
*   **Vibe**: Calm, premium, trustworthy, and modern. Unlike cold fintech, Own Up uses **natural, earthy tones** (greens, teals, yellows) to feel warm and accessible like a friendly advisor's office.

### What Makes It Stand Out
- The **P22 Mackinac serif** paired with a geometric sans is a brilliant typographic choice — it feels both sophisticated and approachable. Very few mortgage brands do this.
- The **mango yellow** (`#FFE485`) as an accent against deep ocean backgrounds creates stunning contrast.
- Uses Gatsby.js (static-rendered React) for a very fast, polished user experience.

### Takeaway for Mortgage by Q
- **This is the closest reference palette to what Mortgage by Q should target.** The earthy ocean-to-forest color story, the serif + sans pairing, and the warm neutral backgrounds translate perfectly to a supportive, trustworthy mortgage advisor.
- Consider a palette variation:
  - **Background**: Very light sage or off-white cream
  - **Accent**: Warm gold or mango
  - **Primary brand**: Deep teal or ocean navy
  - **CTA buttons**: Sage green or deep teal

## 3. Mojo Mortgages (mojomortgages.com) — UK Digital Broker
**URL**: [mojomortgages.com](https://www.mojomortgages.com/)  
**Design Tier**: Tier 2 — Premium Modern Lifestyle

### Design Breakdown
*   **Color Palette**: Bold purple/indigo + bright coral accents + clean whites
*   **Typography**: Strong, custom sans-serif headings with very generous whitespace
*   **Key Features**:
    *   **Conversational UX flow**: Step-by-step "mortgage-ready" questionnaire that feels like a chat app, not a bank form
    *   **Trustpilot integration**: Reviews displayed prominently throughout
    *   **Illustrated characters**: Unique custom illustrations give a friendly, human feel
*   **Vibe**: Playful, bold, and deeply trustworthy. It feels like a consumer app from a forward-thinking fintech startup.

### Takeaway for Mortgage by Q
- The conversational step-by-step UX is highly effective for reducing intimidation. A multi-step "Get My Rate Strategy" wizard (rather than a traditional form) could transform our lead capture rate.

## 4. Movement Mortgage (movement.com)
**URL**: [movement.com](https://movement.com/)  
**Design Tier**: Tier 2 — Premium Modern Lifestyle

### Design Breakdown
*   **Distinctive Feature**: Award-winning use of **"creative image cut-outs"** in layouts. Subjects break out of rectangular frames, creating a dynamic, editorial magazine feel.
*   **Color Palette**: Warm reds + deep blues + clean whites
*   **Typography**: Beautiful custom typography with a human, editorial feel
*   **Vibe**: Community-driven, mission-focused, warm, and dynamic. Feels like an editorial brand, not a bank.

### What Makes It Stand Out
Movement Mortgage has been featured on design award platforms for breaking industry molds. The homepage feels more like a **brand editorial** than a mortgage application portal — a bold strategic bet that helps them attract the next generation of borrowers.

### Takeaway for Mortgage by Q
- Consider editorial-style photography (subjects overlapping frames, bold crops) rather than standard rectangular photos
- A mission-driven "Why We Do This" section humanizes the brand enormously

## 5. Habito (habito.com) — UK Digital Broker (now Monzo-owned)
**URL**: [habito.com](https://www.habito.com/)  
**Design Tier**: Tier 1 — 3D/Interactive Pioneer

### Design Breakdown
*   **Groundbreaking Design History**: Habito was one of the first mortgage brands to use **scroll-triggered animations**, **3D illustrated characters**, and a **fully branded visual universe** (including a campaign featuring a supernatural mortgage villain representing "mortgage hell")
*   **Color Palette**: Warm coral, purple, off-white — a very distinctive non-traditional palette for financial services
*   **Key Features**:
    *   **3D Illustrated Characters**: Custom 3D avatars and illustrated environments bring the brand to life
    *   **Motion-first Design**: Rich CSS/SVG animations that respond to scrolling and user interaction
    *   **Campaign-driven Storytelling**: Famous for their "Mortgage Hell" campaign, which used a 3D demon character to dramatize the pain of getting a mortgage
*   **Vibe**: Whimsical, bold, deeply memorable, and human. Habito proved that a mortgage brand can have a genuine personality.

### What Makes It Stand Out
Habito is the gold standard for **injecting character and personality into an otherwise commoditized financial service**. Their use of 3D and motion proved that mortgage brands don't have to be boring.

### Takeaway for Mortgage by Q
- Even one hero section with a tasteful 3D element (e.g., a floating 3D house model, an animated key icon, or a subtle parallax effect on a home photo) would set the website completely apart from local competitors.
- Consider a bold campaign concept: dramatize the "mortgage stress" problem and position Mortgage by Q as the solution with personality.

## 6. Key Design Technologies Being Used

| Technology | What It Does | Who Uses It |
| :--- | :--- | :--- |
| **Spline.design** | Browser-based 3D scene editor. Embed interactive 3D objects directly in web pages without code | Modern fintech brands for hero elements |
| **Three.js / React Three Fiber** | Full WebGL 3D in the browser. Powers interactive 3D environments | High-budget agency-built sites |
| **GSAP (GreenSock)** | Gold standard for scroll-triggered animations and timeline sequences | Movement Mortgage, Habito, premium sites |
| **Framer Motion** | React animation library for smooth micro-interactions | Fintech startups using Next.js |
| **Astro.js** | Performance-first framework with component islands. What Figure uses | Modern content + marketing sites |
| **Lottie** | Lightweight JSON-based animation format (exports from After Effects) | Mobile-friendly animated icons in hero sections |

## Actionable Design Priority List for Mortgage by Q

### High Impact / Low Effort
1.  **Color Palette Upgrade**: Adopt Own Up's ocean palette variant (deep teal + warm mango/gold + soft sage)
2.  **Typography Pairing**: Use a geometric sans (like `Instrument Sans` or `Outfit`) for body + an approachable display font for headlines
3.  **Hero Section**: Gradient background (deep teal → soft sage) with a floating phone UI mockup overlay

### High Impact / Medium Effort
4.  **Conversational Rate Wizard**: A 3-step "Get My Rate Range" mini-tool on the homepage that doesn't require personal info upfront
5.  **Animated Statistics**: Scroll-triggered counters (e.g., "150+ lenders checked" animating from 0)
6.  **Editorial Photography**: Creative image cut-outs and non-rectangular image frames (inspired by Movement Mortgage)

### High Impact / High Effort (Phase 2)
7.  **Spline 3D Hero Element**: A simple, interactive 3D house or key that users can rotate. Created in Spline, embedded in the homepage hero with no coding required
8.  **GSAP Scroll Animation**: Sections that reveal on scroll with smooth, cinematic transitions
9.  **Dark Mode Option**: A toggle that shifts the site to a deep navy/charcoal premium dark mode
